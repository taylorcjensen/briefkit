## How it works

<Callout type="tip" title="Main workflow">
Run `briefkit dev ./report`. Briefkit opens the browser itself, starts live reload, and keeps generated Astro files out of the report folder.
</Callout>

## Hidden complexity

The report folder stays small. Briefkit creates a disposable Astro workspace under `/tmp/briefkit/` and recreates it whenever needed.
