<BriefTable caption="Hard facts" stickyHeader stickyFirstColumn>

| Topic | Decision |
|---|---|

| Output | Static site |
| Core | One shared installed package |
| Local reports | Thin folders with content, data, components, and public assets |

</BriefTable>

## Inline explanation

A <Tooltip term="throwaway report">A local, purpose-built static site meant to communicate one decision or research result, then be archived or deleted.</Tooltip> should be easy to make and easy to discard.
